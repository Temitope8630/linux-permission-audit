# 🔐 Linux Permission Audit on Kali Linux

## 📌 Project Overview

This project demonstrates a **Linux File Permission Audit** on Kali Linux, focusing on how system permissions are analyzed, identified, and corrected to improve security.

A permission audit is a critical security process used to verify that files and directories have appropriate access controls, ensuring compliance with the principle of **least privilege** and reducing the risk of unauthorized access or privilege escalation.

---

## 🎯 Objectives

- Understand Linux file permissions and ownership
- Learn how to audit system permissions manually and automatically
- Identify insecure permission configurations
- Apply corrective security measures using `chmod`, `chown`, and system tools
- Strengthen system hardening on Kali Linux
